Creational
    - шаблоны для создания объектов класса

    1. abstract factory
        создание семей из разных объектов

        P1 - (A1,B1)
                       -  Factory - (FA,FB)
        P2 - (A2,B2)


    2. builder
        разделение построение сложного объекта позволяя 
        из этих построений создать процессы 
        ( когда часть скрыта)

        P + _Builder - BuilderA - Creator

    3. factory method
        полезен когда у нас одинаковый способ создания 
        нескольких объектов.
        
        Р - (A,B) - creator - (Creator) = Creator
        
    4. prototype
        когда у нас есть множество классов наследуемых 
        от родителя, и мы указываем родителя

        P - (A,B,C) = P

    5. singleton
        один класс для удобнойй работы с объектом

        P  =  p

      работать с ней

    1. adapter

    2. brigde

    3. composite

    4. decorator

    5. facade

    6. flyweight

    7. proxy
